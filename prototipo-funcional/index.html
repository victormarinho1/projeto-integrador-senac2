Sobre o Projeto Integrador

Protótipo desenvolvido para o Projeto Integrador — Centro Universitário Senac (2025). Objetivo: oferecer uma plataforma acessível e segura para a denúncia de trabalho infantil, preservando o anonimato e conectando denúncias aos órgãos competentes (Conselho Tutelar, MPT).

Principais funcionalidades

  • Registro de denúncias com ou sem identificação.
  • Geolocalização do local da ocorrência (simulada no protótipo).
  • Upload de evidências (imagens, vídeo, áudio) com remoção de metadados.
  • Painel administrativo para triagem e acompanhamento.

Tecnologias sugeridas

Backend: Java + Spring Boot • Frontend: Angular • Banco: PostgreSQL • Conteinerização: Docker
